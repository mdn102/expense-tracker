> Full stack expense tracker (MERN)

## [Screenshot]()


### Setup DB
```
# Add config.env file in config folder: 
NODE_ENV=production
PORT=5000
MONGO_URI=fill in your url of mongodb at here
SKIP_PREFLIGHT_CHECK=true
```

### Usage

```
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
